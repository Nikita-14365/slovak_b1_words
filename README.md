# Тренажёр словацкого
Тренажёр для запоминания словацких слов для уровня B1.

# Как пользоваться
Всё просто, запускаете программу, видите сверху слово на русском, внизу надо вписать то же слово на словацком, затем надо нажать клавишу enther (или кнопку check), и, если слово введено правильно, то фон поля ввода текста окрасится в зелёный, а если нет, то в красный, и под ним высветится правильный ответ. Что бы перейти к следующему слову, нажмите ещё раз enther, или кнопку next. Изначально будут показываться только десять случайно выбранных слов, как только для какого-то слова набирается десять правильных ответов, за вычетом неправильных, слово больше показываться не будет, а его место займёт другое слово. Изменить статистику правильных ответов можно вручную внеся изменения в базу данных.

# Зависимости
- python3
- tkinter
- словацкая раскладка

# Как запустить
- скачать репозиторий
- установить python через оффициальный сайт, или через pkg (```pkg install python311```)
- установить tkinter ```python3.11 -m pip install tkinter```
- запустить ```python3.11 main.py``` и пользоваться.
- если чего-то не работает, например, не устанавливается tkinter, спросите о своей трудности на форуме или у чата gpt.
